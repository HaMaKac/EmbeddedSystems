# EmbeddedSystems

### PROJECT TITLE 
Smart Alarm Clock


### DESCRIPTION
The alarm clock with a display and a buzzer playing a melody which can be turned off using different sensors.

 
### DEVICE 
Arduino Uno Rev3 

 
### FUNCTIONALITIES

1.	RTC
2.	Buzzer
3.	Remote control
4.	LCD display
5.	I2C
6.	Temperature and humidity sensor 
7.	Water level detection sensor
8.	Ultrasonic distance sensor
9.	Diodes
